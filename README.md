# MUPYSERVER 
An extremely basic python package server.

## Intro
This is a tiny and simple python package server. Its just for me to be able to `pip install` or `uv add` my own projects locally without having to upload them anywhere.

To build the image do 

```bash
make build
```

To run the image from here do 

```bash
make start
```

and then `make stop` to stop.
